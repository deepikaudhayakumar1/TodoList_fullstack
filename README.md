# 📝 To-Do List Full Stack Application

## 📌 Description
This is a full stack To-Do List application that allows users to manage their daily tasks efficiently. Users can add, update, delete, and mark tasks as completed.

## 🚀 Features
- Add new tasks
- Update existing tasks
- Delete tasks
- Mark tasks as completed
- User-friendly interface

## 🛠️ Technologies Used
- Frontend: React
- Backend:  Spring Boot
- Database: MySQL
